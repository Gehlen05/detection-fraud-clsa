def determinar_periodo_do_dia(hora):
    if 0 <= hora < 6:
        return 'Madrugada'
    elif 6 <= hora < 12:
        return 'Manha'
    elif 12 <= hora < 18:
        return 'Tarde'
    else:
        return 'Noite'

Regressão Logística¶

Os hiperparmetros utilizados no modelo abaixo foram variados e essa foi a configurao final: max_iter=100, ela foi variado para mais e menos e esse foi um balano encontrado sem muitas iteraes para no gerar overftng e nem to poucas pro underfiting; random_state=42, para garantir a reprodutibilidade dos treinamentos; class_weight='balanced', hiperparmetro para melhorar treinamento de datasets desbalanceados; C=0.1, para evitar o risco de overfiting; solver='liblinear'funcionou melhor para o dataset, por ser um dataset menor e baixa dimensionalidade.

Ajustes de Treshold¶

Para encontrar o threshold foram utilizados dois mtodos, o primeiro como um vetor variando o mesmo e dando a pontuao dde F1 e recall. E o segundo a mtrica geomtrica que maximiza a deteco de Verdadeiros Positivos e minimiza os Falsos Positivos. Foi escolhido do segundo mtodo um valor de 0.496, mas casos em prximas discusses for escolhido por levar em considero diminuir um campo pode ser alterado. Aqui tambm as mtricas condizem com oque encontrada na literatura.
